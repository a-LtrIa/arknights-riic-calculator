## 1. 基建总览

### 解锁条件

| 属性 | 数值 |

### 无人机充能速度计算

**基础充能速度** = 1架 / 6分钟

**实际充能速度** = 基础速度 × (1 + 发电站干员加成 + 干员技能加成)

**每日产出无人机** = 240 × (1 + 总加成比例)

### 仓库系统

- 每种产品体积独立计算
- 产品完成后需手动收取，不收取则占用仓库空间
- 仓库满后无法继续生产

### 制造效率计算

**实际耗时** = 基础耗时 / (1 + 效率加成)

**每小时产出** = 60 / 实际耗时(min) × 产出数量

## 12. 进驻机制

### 座位制

- 每个设施有固定座位数，取决于设施等级
- 干员进驻为"坐座位"，换位时重置暖机技能
- 同一设施内干员技能可叠加（加法叠加为主）

### 暖机技能

- 部分干员技能需要暖机时间才能达到最大效果
- **换位重置暖机**：干员离开座位后暖机进度清零
- 典型暖机干员：空构（发电站）、部分贸易站干员

### 注意力涣散

- 干员心情降至0时进入"注意力涣散"状态
- **涣散状态下技能完全失效**
- 需要送回宿舍休息恢复

### 队列轮换与干员休整

- 支持队列轮换：预设替补干员，自动替换涣散干员
- 干员休整：批量管理干员休息安排

## 13. 效率计算汇总

### 核心公式

```
实际生产时间 = 基础耗时 / (1 + 效率加成)
每日产出 = 24 / 实际生产时间(h) × 单次产出
无人机加速效果 = 1架 = 减少3分钟基础耗时
```
